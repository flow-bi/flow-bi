# references/index.md

이 폴더는 특정 라이브러리/툴의 최신 사용법을 압축 요약한 참고자료(llms.txt 스타일)를 모아둔다. AI가 매번 웹 검색을 하지 않고도 이 프로젝트가 실제로 채택한 버전/방식에 맞는 코드를 바로 쓸 수 있게 하는 것이 목적이다.

**주의**: 라이브러리는 계속 업데이트된다. 이 문서에 적힌 API가 실제 설치된 버전과 다르면, 이 문서보다 실제 `package.json`/공식 문서를 우선한다. 버전이 크게 달라졌다면 이 문서를 갱신할 것.
