# 전역 레이아웃 Product Spec

## 기능 목적

서비스 전반에서 공통으로 사용하는 헤더와 대표 색상 선택 기준을 정의한다.

## 상세 요구사항

### 포함한 요구사항

- 헤더 좌측 상단에 기업명을 표시한다.
- 헤더 우측 상단에 본인 이름을 표시한다.
- 헤더 우측 상단 검색창은 존재 가능성이 언급되어 있으나 용도가 확정되지 않았다.
- 대표 색상 테마를 선택할 수 있도록 한다.
- 후보 테마는 흰색/하늘색(`#EDF1F6`), 보라색/흰색/연보라(`#5541A4`)이다.

### 제외한 요구사항

- 각 화면의 본문 구성은 해당 기능별 Product Spec에 둔다.
- 검색 기능의 상세 동작은 원문에 확정되어 있지 않으므로 포함하지 않는다.

## 화면 구성 요소

- 공통 헤더
  - 기업명
  - 본인 이름
  - 검색창 후보 영역
- 테마 선택 요소
  - 흰색/하늘색 테마
  - 보라색/흰색/연보라 테마

## 데이터 흐름

1. 인증된 사용자의 이름을 헤더에 표시한다.
2. 사용자가 테마를 선택하면 서비스 전역의 대표 색상 기준으로 사용한다.
3. 검색창은 용도 미확정 상태이므로 데이터 흐름을 확정하지 않는다.

## 확정된 기능 결정

- 헤더는 모든 주요 화면에 공통으로 존재하는 전역 요소로 본다.
- 기업명과 본인 이름은 헤더의 필수 표시 요소로 본다.
- 대표 색상 선택은 전역 UI 설정으로 본다.

## 확정된 기능 결정의 근거

- `ui-ux-spec.md` 1-0에서 헤더의 좌측 기업명, 우측 본인 이름을 명시한다.
- `ui-ux-spec.md` 1-9에서 대표색상과 선택 가능한 색상 조합을 명시한다.

## Scope Decision

- In Scope: 헤더 표시 정보, 전역 테마 선택 후보.
- Out of Scope: 검색 기능 상세, 개별 화면 레이아웃, 사용자별 테마 저장 방식.
- Moved: 사용자 이름의 원천 데이터는 `my-page.md` 및 `organization-chart.md`와 관련되지만, 헤더 표시는 본 Spec에 둔다.

## Reference Mapping

- `docs/references/ui-ux-spec.md`: 1-0 헤더, 1-9 대표색상

## 미확정 사항

- 검색창의 실제 용도와 검색 대상이 정의되어 있지 않다.
- 테마 선택값의 저장 범위가 사용자별인지 전역인지 정의되어 있지 않다.
