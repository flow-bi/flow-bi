# 대시보드 Product Spec

## 기능 목적

사용자가 로그인 후 오늘과 이번 주의 업무 일정을 빠르게 파악하고 팀원 상태를 확인할 수 있도록 한다.

## 상세 요구사항

### 포함한 요구사항

- AI가 오늘 일정을 줄글로 정리해서 알려주는 영역을 제공한다.
- 주간 일정을 표시한다.
- 오늘의 일정을 표시한다.
- 팀원 상태 영역에는 로그인한 사용자의 주 소속 팀 구성원을 표시한다.

### 제외한 요구사항

- 일정 생성, 수정, 삭제는 `calendar.md`로 이동한다.
- AI와의 대화형 질의응답은 `ai-assistant.md`로 이동한다.
- 팀원 상세 정보 조회는 `organization-chart.md`로 이동한다.

## 화면 구성 요소

- 오늘 일정 AI 요약 영역
- 주간 일정 영역
- 오늘의 일정 영역
- 팀원 상태 영역

## 데이터 흐름

1. 로그인한 사용자의 일정 정보를 기준으로 오늘 일정과 주간 일정을 조회한다.
2. 오늘 일정 정보는 AI 요약 영역에 줄글 형태로 표시된다.
3. 로그인한 사용자의 주 소속 팀 구성원과 상태 정보를 조회한다.
4. 조회한 팀원 상태 정보를 팀원 상태 영역에 표시한다.

## 확정된 기능 결정

- 대시보드는 일정과 팀원 상태를 요약해 보여주는 화면이다.
- 오늘 일정 요약은 AI가 생성한 자연어 문장 형태로 표시한다.
- 팀원 상태 영역에는 로그인한 사용자의 주 소속 팀 구성원을 표시한다.
- 다른 팀 및 전체 조직의 구성원 정보는 `organization-chart.md`에서 조회한다.

## 확정된 기능 결정의 근거

- `ui-ux-spec.md` 1-2에서 AI 오늘 일정 요약, 주간 일정, 오늘의 일정, 팀원 상태를 대시보드 구성으로 명시한다.
- `project-detail.md`의 해결 목표에서 일정 통합 관리와 일정 리마인드 제공을 목표로 제시한다.

## Scope Decision

- In Scope: 일정 요약 표시, 오늘/주간 일정 표시, 주 소속 팀 구성원의 상태 표시.
- Out of Scope: 일정 CRUD, AI 채팅, 조직도 탐색, 다른 팀 및 전체 조직 구성원의 상태 표시, 알림 정책 설정.
- Moved: 일정 상세 작업은 `calendar.md`, AI 대화는 `ai-assistant.md`, 다른 팀 및 전체 조직의 직원 정보 조회는 `organization-chart.md`로 이동한다.
