PR 생성 전 Checklist

브랜치
- [ ] 현재 브랜치가 main이 아니다.
- [ ] 변경 사항이 존재한다.
- [ ] 원격 브랜치가 존재한다.

Issue
- [ ] 관련 Issue가 연결되어 있다.
- [ ] Closes/Related가 올바르다.

Plan
- [ ] Active Plan과 작업 내용이 일치한다.
- [ ] 완료 조건을 만족한다.

검증
- [ ] Build 결과 확인
- [ ] Test 결과 확인
- [ ] Lint 결과 확인

문서
- [ ] PR Template을 사용했다.
- [ ] 변경 사항을 요약했다.
- [ ] Breaking Change 여부를 확인했다.

GitHub
- [ ] 같은 브랜치의 열린 PR이 없다.
- [ ] 사용자 승인을 받았다.
